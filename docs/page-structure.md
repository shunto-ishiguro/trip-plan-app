# ページ構成

## 技術スタック

| プラットフォーム | 技術 |
|-----------------|------|
| Web | Next.js |
| Mobile | Expo (React Native) |

---

## 1. トップ / 旅行一覧ページ（Trip List）

### 共通機能
- 作成した旅行の一覧を表示
- 「新しい旅行を作成」ボタン

### Web
- カード形式のグリッドレイアウト（2〜3列）
- ホバーで詳細プレビュー表示
- サイドバーにフィルター・検索

### Mobile
- 縦スクロールのカードリスト（1列）
- プルダウンで更新
- FAB（Floating Action Button）で新規作成
- スワイプで削除・アーカイブ

---

## 2. 旅行作成ページ（Trip Create / Edit）

### 共通機能
- 旅行タイトル
- 行き先
- 日程（カレンダー）
- 人数
- メモ
- 作成／更新ボタン

### Web
- フォームは中央配置、適度な幅で読みやすく
- 日程はインラインカレンダーで範囲選択
- 入力中のリアルタイムバリデーション

### Mobile
- 縦に並んだフォームフィールド
- 日程はネイティブの DateTimePicker またはボトムシート型カレンダー
- キーボード表示時に入力欄が隠れないようスクロール対応
- 画面下部に固定の「保存」ボタン

---

## 3. 行程編集ページ（Schedule Editor）

旅行ごとの中心ページ。

### 共通機能
- 日ごとのスケジュール一覧
- スポット追加（名称、住所、時間、メモ）
- 順番入れ替え
- スポットの編集・削除

### Web
- 左に日付タブ、右にスケジュール詳細の2カラムレイアウト
- ドラッグ&ドロップで順番入れ替え（react-beautiful-dnd等）
- インライン編集対応
- キーボードショートカット対応

### Mobile
- 上部に日付の横スクロールタブ
- スポットカードの長押しでドラッグ&ドロップ（react-native-draggable-flatlist）
- スワイプで編集・削除メニュー表示
- 「+」FABでスポット追加モーダル

---

## 4. マップページ（Map View）

### 共通機能
- スポットをピンとして表示
- 現在の行程順のルート表示
- 距離・時間の目安

### Web
- 大きな地図表示（画面の70%以上）
- サイドパネルにスポットリスト
- クリックで詳細ポップアップ
- Google Maps / Mapbox 埋め込み
- ルート表示はGoogle Maps Directions APIまたは外部リンク

### Mobile
- フルスクリーンの地図（react-native-maps）
- 下部にスポットリストのボトムシート（スワイプで展開/縮小）
- ピンタップで詳細カード表示
- 「経路を開く」ボタンでOS標準の地図アプリに連携（Apple Maps / Google Maps）
- 現在地表示対応

---

## 5. 予算管理ページ（Budget Manager）

### 共通機能
- 支出項目の追加（交通費、宿泊、食事、その他）
- 合計金額表示
- 1人あたり金額（オプション）
- 項目の編集 / 削除

### Web
- テーブル形式で一覧表示
- インライン編集
- カテゴリ別の円グラフ / 棒グラフ
- CSVエクスポート機能

### Mobile
- カード形式のリスト表示
- スワイプで編集・削除
- 上部にサマリーカード（合計・1人あたり）
- カテゴリ別のシンプルな円グラフ
- FABで項目追加

---

## 6. 共有設定（Share Settings）

### 共通機能
- 共有用の URL / QR コード生成
- 閲覧のみ / 編集可 の権限設定

### Web
- モーダルまたは専用ページ
- URLコピーボタン（クリックでクリップボードにコピー）
- QRコード表示（qrcode.react）
- Web Share API対応ブラウザでは「共有」ボタン表示

### Mobile
- ボトムシートまたはフルスクリーンモーダル
- URLコピーボタン（expo-clipboard）
- QRコード表示（react-native-qrcode-svg）
- 「共有する」ボタンでOS標準の共有シート（Share API）
  - LINE、メール、AirDrop等に直接共有可能
- 権限設定はラジオボタンまたはセグメントコントロール

### 7. チェックリストページ（Packing & To-Do List）

#### 共通機能
- 荷物リスト
- 出発までのやることリスト
- チェックボックスで完了管理

#### Web
- 2カラム（荷物 / やること）
- ドラッグ&ドロップで並び替え
- カテゴリ別折りたたみ

#### Mobile
- タブで切り替え（荷物 / やること）
- チェックボックスは大きめのタップ領域
- 完了項目は下部に移動 or 非表示オプション
- 振動フィードバック（Haptics）でチェック時の触感

---

### 8. 予約情報ページ（Reservation Info）

#### 共通機能
- 航空、ホテル、レンタカーなどの予約情報を登録
- 予約番号、時刻、リンク、メモ
- 当日まとめて確認できる

#### Web
- カード形式で予約種別ごとに表示
- 予約番号はワンクリックでコピー
- 外部リンクは新規タブで開く

#### Mobile
- 予約カードのリスト表示
- 予約番号の長押しでコピー
- リンクはアプリ内ブラウザ or 外部ブラウザで開く
- カレンダーアプリへのエクスポート対応

---

### 9. 設定ページ（Settings）

#### 共通機能
- ユーザー設定
- テーマ切り替え
- 通知設定（任意）

#### Web
- セクション分けされた設定フォーム
- トグルスイッチ / ドロップダウン

#### Mobile
- ネイティブスタイルの設定リスト（iOS: grouped、Android: Material Design）
- テーマ切り替えは即時プレビュー
- プッシュ通知の許可リクエスト（expo-notifications）
- アプリ情報・バージョン表示
